MEAPI (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates using the MEAPI to receive available exchanges, 
products, and contract definitions.


Instructions
-------------------------------------------------------------------------------

1. Click the "Request Gateways" button at the bottom of the window.
2. Select an exchange from the list
3. Select a product type from the list - products should now appear.
4. You can drag and drop individual contracts from this sample to X_TRADER and
   supported XTAPI applications.


MEAPI Objects
-------------------------------------------------------------------------------

TTMarketExplorerClass
ITTGateway
ITTProductCollection
