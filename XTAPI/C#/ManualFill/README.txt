ManualFill (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates how to create manual fills using the XTAPI.  This 
sample requires you have administrative risk privileges.


Instructions
-------------------------------------------------------------------------------

1. Login with administrator credentials
2. Drag a contract onto the form from the Market Grid
3. Enter a price, quantity, format and side
4. Click "Create Manual Fill"


XTAPI Objects
-------------------------------------------------------------------------------

TTRiskManager
TTManualFill
TTGate
TTDropHandler
TTInstrObj
TTInstrNotify
